Origins

taiyoML began as a study of applied machine learning, testing the feasibility of live photovoltaic output predictions on the cloud.

Future

Now, taiyoML is a running experiment in bringing the esoteric domain of machine learning to users in a simple intuitive interface. Still committed to simplicity, the new interface aims to be the leading example in cloud based machine learning.
